      # Introdução

      Introdução¶

      xxxxxxxxxx

      Criam-se processos `R` e `Python` com tempo de execução não simultâneos entre si. Esse modelo é satisfatório para um bom aproveitamento.
      ​
      Assim, quando um programa de código `R` tiver sua execução concluída, sua sessão iterativa será pausada para que o processo iterativo `Python` continue.

      Assim, quando um programa de código R tiver sua execução concluída, sua sessão iterativa será pausada para que o processo iterativo Python continue.

      xxxxxxxxxx
       
      O destino padrão do **output textual** das funções `print` e `cat` no `R` será o terminal. Pode-se programar o desvio para o `jupyter-notebook` ou outra interface interativa. Isso pode ser feito desviando-se o resultado para um arquivo *.txt* temporário e finalmente para a interface interativa.
      ​
      Uma estratégia análoga pode ser utilizada para o desvio de resultados em **imagens**, que são, por padrão, abertas em dispositivos interativos (janelas).
      ​
      Analogamente, pode-se utilizar arquivos *txt* temporários como intermediários de **interconversão de variáveis** por meio do formato *data frame*. A ideia é hospedar uma variável de um processo em um arquivo temporário *txt*. Pode ser interessante armazenar a variável nesse arquivo pelo formato *data frame*, que seria um formato intermediário entre os processos. 
      ​
      Ou seja, a comunicação de resultados dos processos pode ocorrer por meio de arquivos temporários e mecanismos de importação e exportação.
      ​
      Essas estratégias podem ser automatizadas pelo usuário, sobretudo conhecendo-se as técnicas a seguir.

      def r_input(lines):
          new_lines = []
          for line in lines:
              chars=line
          # ~~~~~~~~~~~~~~~~~~~~~~~~~#
              if(chars=="%r\n"): # Abertura do código em R / Sinalização
          # ~~~~~~~~~~~~~~~~~~~~~~~~~#
                  chars='''text="""\n'''
              else:
              # ~~~~~~~~~~~~~~~~~~~~~~~~~#
                  if(chars=="r%\n"): # Fechamento do código em R / Sinalização
              # ~~~~~~~~~~~~~~~~~~~~~~~~~#
                      chars='''"""\n'''
                      chars=chars+"reval(text)\n"
              new_lines.append(chars)
          return new_lines
